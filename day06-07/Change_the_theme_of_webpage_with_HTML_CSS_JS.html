説明

themeColor.html
下部ボタンのonclickでそれぞれテーマを変える関数を実行する

themeColor.css
4~23 : 変数--color-なんとかでそれぞれのテーマの色を指定

themeColor.js
1~6 : まずlocalStorageを見に行って値が存在しない(Null)場合は初期化 (changeBlue()を実行)
存在する場合はキーLSthemeの値をルート(<html>)のtheme属性の値として設定する

10~23 : テーマを変更する関数
2行目でlocalStorageに値をセット
3行目でルートの属性と値を設定
